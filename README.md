# Task Manager

A simple port manager application built by @rakibtg

The goal of this project is to create a cross-platform port manager application to help users manage their network ports easily.

## Getting Started

To get started with the development environment, follow these steps:

```bash
npm install
npm run tauri android init

# For Desktop development, run:
npm run tauri dev

# For Android development, run:
npm run tauri android dev
```

## Features

- List all active network ports
- Start and stop services on specific ports
- Cross-platform support (Desktop and Android)
- User-friendly interface
